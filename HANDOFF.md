# Translation Rules

Convert GitBook syntax into Docusaurus-compatible Markdown/MDX.

Required conversions:

-   GitBook frontmatter → Docusaurus frontmatter

-   .gitbook/assets → /gitbook/`<app>`{=html}/

-   ```{=html}
    <figure>
    ```
    → Markdown image

-   {% hint %} → Docusaurus admonitions

-   {% stepper %} → Docusaurus-compatible markdown/MDX

-   {% tabs %} → Docusaurus Tabs

-   {% content-ref %} → Markdown links

-   {% expand %} →
    ```{=html}
    <details>
    ```

-   {% embed %} → supported embed

Do not modify GitBook repositories.

Only translated output goes into docs/.
