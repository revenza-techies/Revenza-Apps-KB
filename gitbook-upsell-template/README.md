# Revenza Upsell Content

This repository is synced with the GitBook Revenza Upsell space and stores editable app documentation for `docs.revenza.in/revenza-upsell`.

## Safe to edit in GitBook

- `intro.md`
- `faq.md`
- `getting-started/**/*.md`
- `offers/**/*.md`
- `customization/**/*.md`
- `troubleshooting/**/*.md`
- `images/**/*`

## Image links

Upload images under `images/`, then reference them in Markdown using the final Docusaurus public path:

```md
![Install Revenza Upsell app](/img/content/revenza-upsell/getting-started/install-app.webp)
```

Use lowercase filenames with hyphens and avoid screenshots with customer data, secrets, or private store information.
