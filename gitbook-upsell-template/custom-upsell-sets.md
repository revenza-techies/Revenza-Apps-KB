# Custom Upsell Sets

Custom upsell sets let you choose which products should be recommended on specific product pages.

## Create a custom set

1. Open Revenza Upsell in Shopify admin.
2. Go to **Custom Upsell Sets**.
3. Create a new set.
4. Add the products you want to recommend.
5. Save the set.

Use clear set names so you can manage offers easily as your store grows.
