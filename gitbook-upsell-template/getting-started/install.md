# Install

Install Revenza Upsell app from Shopify App Store.

## Steps

1. Open the Revenza Upsell listing in the Shopify App Store.
2. Select **Install**.
3. Review the requested permissions.
4. Complete the installation in Shopify admin.
5. Open Revenza Upsell from your Shopify apps list.

:::tip
Use the same Shopify store where you want the upsell block to appear.
:::
