# Text and translations

Use short, specific language that tells customers what is being offered and what happens next.

## Recommended copy pattern

- Heading: explain why the products belong together.
- Supporting text: add only information needed to make a decision.
- Button: use a clear action such as **Add selected items**.

Test translated copy at mobile width because longer labels may wrap.
