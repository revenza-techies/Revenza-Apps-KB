# Mapping

Mapping connects a custom upsell set to one or more Shopify products.

## Map a set

1. Open the set you want to display.
2. Choose the product or products that should trigger the set.
3. Save the mapping.
4. Preview the mapped product on your storefront.

If a product does not show the expected set, confirm the app embed is active and the product is included in the mapping.
