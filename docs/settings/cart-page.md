# Cart page

#### Enable Cart Page Widget

Enable or disable the upsell widget on the cart page.

* **Enabled:** Displays upsell products on the cart page.
* **Disabled:** Hides the widget from the cart page.

> **Note:** The widget must also be integrated into your theme using the App Block or App Embed.

***

#### Heading

Enter the title displayed above the upsell products.

**Example:** _Also Buy_, _You May Also Like_, or _Recommended for You_.

***

#### Priority to Cart Item Mappings

When enabled, the app checks the products currently in the customer's cart and gives priority to any upsell sets mapped to those products.

* **Enabled:** Displays the mapped upsell set for the cart items.
* **Disabled:** Uses the default upsell set when no other rules apply.

> **Note:** This setting only works if you have created product mappings.

***

#### Default Upsell Set

Select the default upsell set to display on the cart page.

This set is used when no mapped upsell set is available for the products in the cart.

***

#### Product Count

Choose how many upsell products to display.

* Minimum: **1**
* Maximum: **20**

Select a number that fits well within your cart layout.

***

#### Card Style

Choose the visual style of the product cards displayed in the cart upsell widget.
