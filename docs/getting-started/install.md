# Install Revenza Upsell

Use this checklist to connect Revenza Upsell to your Shopify store.

## Before you begin

- Sign in to the Shopify admin for the store you want to configure.
- Confirm your account can install apps and update theme app embeds.
- Keep a test product ready so you can preview the storefront experience.

## Installation steps

1. Open the Revenza Upsell listing in the Shopify App Store.
2. Select **Install** and review the requested permissions.
3. Complete the installation in Shopify admin.
4. Open Revenza Upsell and follow the onboarding checklist.
5. Enable the app embed or app block when prompted.

:::tip Preview before publishing
Use a test product to confirm the offer matches your theme before showing it to customers.
:::
