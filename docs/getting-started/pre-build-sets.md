# Pre-build sets

Open any product in your store to check the Revenza Upsell app block below the **Add to Cart** button.

## Check the storefront

1. Open your storefront.
2. Visit any product page.
3. Confirm the Revenza Upsell block appears below the **Add to Cart** button.
4. Review the default offer layout on desktop and mobile.

The app is ready for use when the block is visible and the storefront layout looks correct.
