# Getting Started

The setup takes less than a minute.

### Step 1: Open the Theme Editor <a href="#step-1-open-the-theme-editor" id="step-1-open-the-theme-editor" />

Open the **Revenza Upsell** app from apps section. In the **Enable app embed** block, click on **Activate** button.

:::info
**Note:** The button opens your currently published Shopify theme in the Theme Editor.
:::

---

### Step 2: Save the App Embed <a href="#step-2-save-the-app-embed" id="step-2-save-the-app-embed" />

The Theme Editor will open with the **Revenza Upsell** app embed already enabled. Click **Save** in the top-right corner to apply the changes to your live theme.

:::info
**Important:** The app embed is not activated until you save the theme.
:::

---

### Step 3: Verify the Integration <a href="#step-3-verify-the-integration" id="step-3-verify-the-integration" />

Return to the Revenza Upsell app. The **App Embed** status should now display **Enabled**, confirming that the app has been successfully integrated with your live theme.

---

### Step 4: Preview the Upsells <a href="#step-4-preview-the-upsells" id="step-4-preview-the-upsells" />

Click on preview button on Product page block. The default **Pre-built Upsells** will automatically appear on the product pages.
