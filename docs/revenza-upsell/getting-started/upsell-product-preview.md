# Upsell Review

You can preview the Pre-built upsells on the following pages:

### **Product Page Block**

Please use the preview button to review the Pre-build upsells on the Product page.

:::info
In case the Upsell block does not appear on the product page, please [_**integrate the Upsell App Block**_](/revenza-upsell/integration/app-block-integration-product-page) on the product page using the manual integration process.
:::

### **Cart Page Block**

Please use the preview button to review the Pre-build upsells on the Cart page. It is mandatory to add a product in the cart for upsell block visibility on the cart page.

:::info
In case the Upsell block does not appear on the cart page, please [_**integrate the Upsell App Block**_](/revenza-upsell/integration/app-block-integration-cart-page) on the cart page using the manual integration process.
:::
