# Overview

Welcome to Revenza Upsell docs

## Grow every order with smarter upsells

Revenza Upsell helps Shopify merchants create relevant product offers that make it easier for customers to discover more items.

### Start here

Follow these steps to launch your first upsell.

<Stepper>
  <Step>
    ### Install

    Add Revenza Upsell to your store.\
    [Install](https://rvus-test.revenza.in/api/v1/auth?shop=revenza-yv.myshopify.com)
  </Step>

  <Step>
    ### Activate App

    Build your first upsell offer.\
    [Enable App](/docs/revenza-upsell/getting-started/)

    ![](/gitbook/revenza-upsell/Image%201.png)
  </Step>

  <Step>
    ### Go Live

    Upsell are now active on the store front.\
    [Pre-built sets](/docs/revenza-upsell/pre-build-upsell-sets)
  </Step>
</Stepper>
