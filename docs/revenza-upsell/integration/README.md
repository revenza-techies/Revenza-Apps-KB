# Integration

## App Block (2.0) - Integration

Use the **App Block** to display upsell products on your **Product** and **Cart** pages without editing your theme code.

This no-code integration method allows you to quickly add upsell widgets to your storefront, making it easier for customers to discover relevant products and encouraging additional purchases.
