# Settings

The **Settings** page lets you control how the Revenza Upsell widget behaves across your Shopify store. Configure separate settings for the **Product Page**, **Cart Page**, and **Global** preferences to customize the shopping experience, control widget appearance, and optimize product recommendations.

From this page, you can:

* Enable or disable upsells for specific pages.
* Customize the widget heading and default product set.
* Choose the number of products to display.
* Select card styles and layout options.
* Configure add-to-cart behavior.
* Apply global display preferences and custom CSS.
