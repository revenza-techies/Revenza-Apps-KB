# Frequently asked questions

## Where do Revenza Upsell offers appear?

Placement depends on the offer type and the app block or embed configured in your Shopify theme.

## Can I preview an offer before publishing?

Yes. Preview every offer on a test product and check the desktop and mobile storefront experience before publishing.

## Will an offer match my theme?

You can customize the offer presentation and use your theme's existing colors, typography, and button styles as a guide.

## How do I suggest a documentation update?

Use **Edit this page** at the bottom of any article. GitHub opens the source file so you can propose a change through a pull request.
