# Revenza Knowledge Base

A centralized documentation platform for Revenza Shopify applications.

Built with **Docusaurus**, authored in **GitBook**, synchronized through **GitHub**, and published via **GitHub Pages**.

---

## Overview

The Revenza Knowledge Base provides merchants with installation guides, feature documentation, troubleshooting articles, FAQs, release notes, and support resources for all Revenza Shopify applications.

GitBook is the single source of truth for documentation, while Docusaurus provides the branded documentation website.

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash

```

Open:

```
http://localhost:3000
```

---

## Documentation Workflow

1. Create documentation in GitBook.
2. Sync GitBook to GitHub.
3. Docusaurus renders the synchronized content.
4. GitHub Pages publishes the latest version.
